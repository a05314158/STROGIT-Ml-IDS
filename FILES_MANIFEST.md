4. Прочитайте speaker_notes.md 2-3 раза

### За день до конференции:
```
1. Проверьте интернет соединение
2. Откройте все файлы в браузере
3. Проверьте звук и микрофон
4. Прочитайте speaker_notes.md ещё раз
5. Подготовьте воду для горла
```

```
1. Приходите за 30 минут до доклада
2. Проверьте проектор и звук
3. Откройте conference_presentation.html
4. Держите speaker_notes.md на втором мониторе
5. Начните доклад с уверенностью!
```
