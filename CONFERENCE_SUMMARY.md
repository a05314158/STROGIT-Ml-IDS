## 🚀 Быстрый старт

### За неделю до конференции:
```
1. Откройте conference_index.html в браузере
2. Прочитайте conference_abstract.md
3. Посмотрите conference_presentation.html
4. Прочитайте speaker_notes.md 2-3 раза
5. Распечатайте conference_handout.html
```

### За день до конференции:
```
1. Проверьте интернет соединение
2. Откройте все файлы в браузере
3. Проверьте звук и микрофон
4. Прочитайте speaker_notes.md ещё раз
5. Подготовьте воду для горла
```

### В день конференции:
```
1. Приходите за 30 минут до доклада
2. Проверьте проектор и звук
3. Откройте conference_presentation.html
4. Держите speaker_notes.md на втором мониторе
5. Начните доклад с уверенностью!
```

### Перед докладом:
- Говорите медленно и чётко

### Управление временем:
- Не спешите - 25 минут достаточно

- Если не хватает - пропустите детали

### Ответы на вопросы:
- Используйте speaker_notes.md как справочник
